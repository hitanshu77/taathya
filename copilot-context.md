
# AI Development Context

This project is a  **real estate property explorer web application** .

The explorer contains:

• Mapbox GL JS 3D skyline
• property card carousel
• geospatial property coordinates
• filtering system (type / bhk / price / location)
• cinematic hero section
• interactive map-camera fly animations

The codebase uses:

• HTML
• CSS
• Vanilla JavaScript
• Mapbox GL JS
• Node scripts for dataset processing

# Coding Standards

When modifying code:

• Prefer clean modular JavaScript
• Avoid unnecessary frameworks
• Optimize for performance and smooth animations
• Ensure Mapbox layers load safely and avoid race conditions
• Maintain readable UI layout structure

---

# Map System Rules

The map must:

• render 3D buildings using fill-extrusion
• sync with the property carousel
• animate camera flyTo when cards change
• only render landmark projects at low zoom
• load additional buildings when zooming in

---

# UI Experience Goals

The website should feel like a  **premium real estate explorer** :

• cinematic hero introduction
• smooth transitions
• luxury dark theme
• glowing 3D skyline
• fluid card interactions

Focus on  **performance, clarity, and visual polish** .
